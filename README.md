# 🍽️ QuickServe — Restaurant Order Management System

QuickServe is a real-time web-based Restaurant Order Management System designed to streamline operations between Waiters, Kitchen Staff, and Billing Counter.

## Features
- **Waiter Flow:** Create and manage orders.
- **Kitchen Flow:** View and process incoming orders in realtime.
- **Counter Flow:** Generate bills and accept payments.
- **Real-time Sync:** Powered by Supabase Realtime, updates sync instantly across devices.

## Tech Stack
- Frontend: React (Vite), TypeScript, Tailwind CSS v4, Zustand, React Query
- Backend: Supabase (PostgreSQL, Auth, Realtime)

## Setup Instructions

### 1. Supabase Setup
- Create a new project in [Supabase](https://supabase.com).
- Copy the provided SQL from `backend/supabase/migrations/01_initial_schema.sql` and run it in the SQL Editor to create tables, policies, and enable Realtime.
- Copy the `backend/supabase/seed.sql` and execute it to create dummy users and test menu items.

### 2. Frontend Setup
- Navigate to the web app:
```sh
cd apps/web
npm install
```
- Create a `.env` file in `apps/web` with:
```env
VITE_SUPABASE_URL=your_supabase_project_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
```

### 3. Run Development Server
```sh
npm run dev
```
